Announcements

Changes in

  • Added better description for Open Door checks
  • Added missing misc modifier for Open Door checks
  • Added Misc Modifier to the three static monster attack buttons
  • Added notes for Head and Touch AC
  • Added notes for Touch-delivered spells
  • Added notes for wishing for ability scores
  • Added reverse spells. All data related to the reverse spell is exactly the same as the original spell. I.e. no special changes have been made specific to the reverse spell.
  • Added inline labels for modifiers to attack, damage, initiative, surprise, saving throw, and magic resistance rolls
  • Corrected color for initiative rolls. Now a roll of 1 will have a green boarder (crit) and a roll of 10 will have a red boarder (fumble)
  • Improved the layout for Player's Option critical hits a bit
  • Fixed cast time for Protection From Evil (Wizard) from 1 turn to +1 initiative modifier
  • Removed incorrect material components for Endure Cold/Endure Heat
  • Fixed spell description layout for Orison
  • Removed Weapon Type vs Armor type modifiers from the Hit roll under the Player's Option ruleset, as the Player's Option: Combat & Tactics has its own set of modifiers that differ from those given in the Player's Handbook
  • Weapon Type vs Armor type is now only visible under the Standard tab
  • Change scroll naming convention from "Scroll of X" to "X scroll". This put the spell name first and makes it easier to read on small screens.
  • Fixed Follower->Psion->Repeating->Standard->Initiative button
