The last Mutant: Year Zero sheet made use of highly unintuitive attribute names, these attribute names primarily posed an issue when a GM attempted to assign a value to a token (e.g. health bars and such). The only way one could reliably figure out what an attribute name represented was to look at the code (For example, the attribute name representing Damage was named "utmattning", which is not only in Swedish, but also means Fatigue)

There were also many cases of attribute names being identical to each other, which resulted in that some input fields would mimic the text or value of other fields.

These issues has been fixed in the new sheet and this tab will eventually be removed in a future update. In this tab you will find any attributes that did not carry over to the new sheet. However, if you wish to save your old sheet it is required that you copy over the values from this tab to the Character Sheet and Ark tab before the set date of removal.

We apologize for the inconvenience, and wish you a merry time in the zone.

The planned removal for this tab is set to be at November 1st, 2015
