Use the checkboxes to indicate whether each row is is worn/equipped (meaning it applies to your normal AC score) and whether the row also provides a benefit to your unarmoured AC score (such as magical items or in some cases a shield). If you have a class feature that also boosts your unarmoured AC then select that here too.

NPC/Monster Sheet F

This page of the character sheet is intended for use by GM/DM's only. All the macros on this page send their output only to the GM via a private whisper/tell. GM/DM's should be sure to make sure any data entered into larger text fields is all on one line as any line breaks will cause data after it to be sent to public chat. GM/DM's can workaround this by adding "/w GM " just after each line break (without the quotes) to ensure the output only goes to them.
